# Documentation Interne

Ce dossier contient la **documentation interne** de DooDates, destinée uniquement à l'équipe de développement.

## ⚠️ Important

Les documents de ce dossier :
- **NE DOIVENT PAS** être exposés publiquement aux utilisateurs finaux
- **NE DOIVENT PAS** être liés depuis l'interface utilisateur
- Sont destinés à un usage interne uniquement (développement, audit, planification)

## Contenu

### RGPD-Audit-Complet.md
Audit interne de conformité RGPD réalisé le 10 décembre 2025. Ce document détaille :
- L'état d'avancement de la conformité RGPD
- Les actions réalisées et restantes
- Les mécanismes techniques implémentés
- Le planning d'automatisation

**Note :** Les utilisateurs ont accès à la politique de confidentialité publique dans `/Docs/LEGAL/Politique-Confidentialite-IA.md`, qui contient les informations nécessaires pour eux.
