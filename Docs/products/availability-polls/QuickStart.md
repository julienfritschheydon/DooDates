# Guide Utilisateur - Sondages de Disponibilité

**Trouvez le moment parfait pour tous.**

## Fonctionnalités
- **Synchronisation** : Identifiez les créneaux communs à tous les participants.
- **Flexibilité** : Indiquez vos disponibilités sur une semaine type ou des dates spécifiques.
- **Rapidité** : Visualisez en un coup d'œil les meilleures options.

## Comment créer un sondage ?
1. Allez sur la page [Sondages de Disponibilité](/availability-polls).
2. Cliquez sur **"Nouveau Sondage"**.
3. Définissez la plage de temps concernée.
4. Partagez le lien.
5. Chaque participant remplit ses disponibilités.
