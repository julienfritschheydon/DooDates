# Guide Utilisateur - Formulaires

**Collectez des informations structurées.**

## Fonctionnalités
- **Questions Variées** : Texte libre, choix unique, choix multiple, etc.
- **Personnalisation** : Adaptez le formulaire à vos besoins.
- **Analyse en Temps Réel** : Visualisez les réponses au fur et à mesure.

## Comment créer un formulaire ?
1. Allez sur la page [Formulaires](/form-polls).
2. Cliquez sur **"Nouveau Formulaire"**.
3. Ajoutez vos questions.
4. Configurez les options de réponse.
5. Partagez le lien pour commencer la collecte.
