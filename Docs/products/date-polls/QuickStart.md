# Guide Utilisateur - Sondages de Dates

**Planifiez vos événements simplement.**

## Fonctionnalités
- **Propositions Multiples** : Suggérez plusieurs dates et créneaux horaires.
- **Vote des Participants** : Invitez vos collègues ou amis à indiquer leurs disponibilités.
- **Consensus IA** : Laissez notre intelligence artificielle analyser les réponses pour trouver le meilleur créneau.

## Comment créer un sondage ?
1. Allez sur la page [Sondages de Dates](/date-polls).
2. Cliquez sur **"Nouveau Sondage"**.
3. Sélectionnez les dates possibles dans le calendrier.
4. Ajoutez des créneaux horaires si nécessaire.
5. Partagez le lien avec vos participants.
